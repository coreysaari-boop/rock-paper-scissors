Rock Paper Scissors
A simple, animated Rock‑Paper‑Scissors game built with HTML, CSS, and JavaScript. Features include score tracking, round history, game‑over modal, selectable game modes, animated UI feedback, and visual progress bars.

✨ Features
Interactive gameplay with Rock, Paper, and Scissors buttons

Best‑of‑5 and Best‑of‑7 modes

Rounds‑to‑win indicator

Animated score updates

Round history log

Game‑over modal with Play Again option

Progress bars showing each player’s path to victory

Clean, responsive layout

🎮 How to Play
Choose Rock, Paper, or Scissors.

Watch the animated result and score update.

First to the required number of wins (based on mode) wins the game.

Click Play Again or Reset Game to start fresh.

Switch modes anytime — the game resets automatically.

🧠 Game Modes
Best of 5 → first to 3 wins

Best of 7 → first to 4 wins

The selected mode is highlighted and displayed in the rounds‑to‑win indicator.

🛠️ Technologies Used
HTML — structure

CSS — animations, layout, styling

JavaScript — game logic, DOM updates, animations
